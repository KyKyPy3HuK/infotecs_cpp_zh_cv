Проект для тестового задания на стажировку Infotecs Academy
by Жупин Святослав (github KyKyPy3HuK) 21.08.23

Проект состоит из двух программ client (программа №1 в задании) и server (программа №2 в задании), исходные коды и Makefile'ы которых находятся в соответствующих директориях
Обе программы собираются средствами make и компилятора gcc (g++)

Процесс сборки:

	Сборка программы-клиента ClientApp.out (необходимо перейти в папку client):
		make       	#сборка
		make debug 	#сборка с информаций для отладки
		make clean 	#удаление объектных и исполняемых файлов
	
	Сборка программы-сервера ClientApp.out (необходимо перейти в папку server):
		make      	#сборка
		make debug 	#сборка с информаций для отладки
		make clean 	#удаление объектных и исполняемых файлов
		
		
Описание работы программ:

	Программа-клиент (Программа №1):
		Работает в двух потоках - поток 1 принимает строку, которую введет пользователь, 
		введенная строка проверяется на то что она состоит только из цифр и не превышает 64 символа. 
		После проверки строка сортируется по убыванию и все четные элементы заменяются на латинские буквы «КВ». 
		После данная строка помещается в общий буфер и поток ожидает дальнейшего ввода пользователя.
		
		Поток 2 обрабатывает данные, которые помещаются в общий буфер. После получения
		данных общий буфер затирается. Поток выводит полученные данные на экран и рассчитывает
		общую сумму всех элементов, которые являются численными значениями. Полученную сумму
		передает в Программу №2. После этого поток ожидает следующие данные.
		
	Программа-сервер (Программа №2):
		Ожидает данные от Программы №1. При получении данных происходит анализ из скольки
		с имволов состоит переданное сообщение. Если оно больше 2-ух символов и если оно кратно 32
		выводит сообщение о полученных данных, иначе выводится сообщение об ошибке (recv Error). 
		Далее программа продолжает ожидать данные.

	std::string replaceNums(std::string message){
    		std::string processedMessage(message);

    		for (int i = 0; i < processedMessage.size(); i++) {
        		if (((processedMessage[i] - '0') % 2) == 0)
        		{
           			processedMessage[i] = 'B';
            			processedMessage.insert(i,"K");
            			i++;
        		}
    		}

	}
